# 商业底线配置问答模板

> 逐模块向用户提问，每个问题提供示例帮助理解。用户可直接回复编号或自定义值。

---

## 模块 1：付款规则

请设定贵司的付款规则（可直接回复示例编号或自定义）：

1. 首款比例要求：____% （示例：30%）
2. 首款支付期限：合同签署后 ____ 日内 （示例：10日）
3. 可接受的最长付款周期：____ 日 （示例：30日）
4. 验收逾期违约金：每天 ____‰ ，上限合同总价 ____% （示例：0.5‰，3%）

## 模块 2：责任上限

请设定贵司的责任边界：

1. 违约责任总额上限：不超过实际收款额的 ____% （示例：100%，即实收费用总额）
2. 赔偿范围限定为：____ （示例：仅限直接经济损失）
3. 不承担的损失类型：____ （示例：间接损失、利润损失、惩罚性损害）

## 模块 3：知识产权

请设定贵司的知识产权策略：

1. 标准产品是否提供源代码？ （示例：否）
2. 定制功能源代码如何处理？ （示例：可提供给客户，IP 归客户）
3. 是否允许"共同拥有知识产权"条款？ （示例：仅限定制/共创功能部分）
4. 产品名称要求：____ （示例：须与软著名称一致）

## 模块 4：产品服务规则

请设定贵司的产品服务规则：

1. 订阅产品未续费是否自动终止？ （示例：是，license 停用）
2. 买断产品免费维保期：____ （示例：1年）
3. 维保费率：按产品金额的 ____% （示例：15%）
4. 买断/定制产品是否提供免费升级？ （示例：否）
5. 项目交付周期上限：____ （示例：6个月）

## 模块 5：保密与数据安全

请设定贵司的保密与数据安全规则：

1. 保密期限上限：____ （示例：项目周期+3年）
2. 是否接受"永久保密"条款？ （示例：否）
3. 数据泄露通知时限：____ 小时内 （示例：24-72小时）
4. 数据安全连带责任是否有上限？ （示例：是，不超过实收费用）

## 模块 6：合规条款

请设定贵司的合规要求：

1. 违反廉洁条款的违约金：合同总价的 ____% （示例：5%-10%）
2. 产品税率：____% （示例：13%）
3. 服务税率：____% （示例：6%）
4. 是否需要廉洁协议附件？ （示例：是）

## 模块 7：服务方权益

请设定贵司的服务方权益保障：

1. 逾期 ____ 日可暂停服务 （示例：30日）
2. 暂停后超过 ____ 日仍未付款可终止合同 （示例：60日）
3. 履约保证金比例范围：合同总价的 ____% - ____% （示例：5%-10%）
4. 联系人变更须提前通知：____ 个工作日 （示例：10日）
