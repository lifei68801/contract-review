# 会话管理实施手册

## 快速参考

### 当前配置
- **上下文限制**: 100K tokens
- **压缩模式**: default（自动）
- **完整会话**: ~/.openclaw/agents/main/sessions/*.jsonl
- **摘要存储**: memory/sessions/{daily,weekly,longterm}/

### 分层架构
```
L1: 最近10轮完整  ~30K  ← 始终保留
L2: 当日摘要      ~10K  ← 每4小时生成
L3: 本周摘要      ~5K   ← 每周日合并
L4: 长期记忆      ~5K   ← 每天23:00更新
────────────────────────────
总计 ≤ 50K (安全余量)
```

### 紧急情况处理

**如果仍出现 context exceed:**

1. **自动处理**（OpenClaw 自动）
   - 超过 100K 时自动压缩
   - 保留最近消息

2. **手动处理**
   ```
   用户："重置对话"
   ```

3. **检查状态**
   ```
   用户："查看上下文状态"
   ```
