# 合同审核关键词速查表

> 审核合同时，用以下关键词在合同文本中搜索，快速定位潜在风险条款。

## 🚫 否决级关键词（出现即需重点审查）

| 关键词 | 风险 | 对应条款 | 处理方式 |
|--------|------|----------|----------|
| 永久保密 / 无期限保密 | 保密期无上限 | R-018 | 修改为项目周期+3年 |
| 共同拥有 / 共同享有知识产权 | 侵犯我方IP | R-004 | 限定为定制功能部分 |
| 全部源代码 / 源码交付 | 源码泄露 | R-002 | 仅限定制功能源码 |
| 连带赔偿 / 无限连带 | 赔偿无上限 | R-006 | 修改为直接损失+责任上限 |
| 利润损失 / 间接损失 | 超范围赔偿 | R-006 | 删除或限定为直接损失 |
| 惩罚性损害 | 超范围赔偿 | R-006 | 删除 |
| 赔偿不封顶 / 无上限赔偿 | 责任无上限 | R-005 | 设置为实收费用总额上限 |
| 免费升级 | 过度承诺 | R-015 | 买断/定开产品不支持 |
| 付款.*超过.*30日 / 付款周期.*60日 | 回款风险 | R-011 | 最长30日 |
| 数据安全.*无限连带 | 数据责任无上限 | R-019 | 设置上限 |
