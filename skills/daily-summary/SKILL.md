# 每日工作总结

## 触发词
- "总结一下今天"
- "今日总结"
- "工作总结"
- "今天干了什么"

## 执行步骤

1. 读取 `~/.openclaw/workspace/memory/daily-log.md`
2. 提取今天的记录
3. 生成简洁总结

## 输出格式

```
📊 今日工作总结

• 要点1
• 要点2
• 要点3

✨ 明天继续加油！
```

## 如果没有记录

```
📝 今天还没有记录哦

随时告诉我"记录：xxx"，我帮你记下来~
```

## 记录方式

用户说"记录：完成了xxx"或"今天做了xxx"，追加到 `daily-log.md` 当天部分。
