# Web Search Plus - 挖到宝了！这个搜索工具太神了！

## 🚀 我的天！搜索还能这么玩？

最近发现个神仙技能 - Web Search Plus，直接刷新我的搜索体验！这可不是普通的搜索工具，是个智能搜索大脑，自己帮你挑最合适的搜索引擎。

## 🧠 它到底有多聪明？

最牛的是**智能自动路由**！你随便输入问题，它自己分析该用哪个：

- 买东西查价格？→ 自动用 Google 搜索（Serper）
- 想做深度研究？→ 自动用 AI 优化搜索（Tavily）  
- 找类似公司？→ 自动用语义搜索（Exa）
- 怕被追踪？→ 自动用无痕引擎（SearXNG）

实际例子：
```bash
python3 scripts/search.py -q "iPhone 16 价格"     # → 自动去 Google 搜
python3 scripts/search.py -q "量子计算怎么学"     # → 自动去 AI 深搜
python3 scripts/search.py -q "类似 Notion 的工具"  # → 自动语义匹配
```

## 🔥 7个搜索引擎随便换

1. **Serper** - 就是 Google 搜索，查价格、找店、看新闻绝了
2. **Tavily** - AI 专用搜索，回答问题、做研究超强
3. **Querit** - 多语言搜索，中日德文等10+语言都行
4. **Exa** - 语义搜索，找相似网站、发现公司神器
5. **Perplexity** - 直接给答案，还带引用
6. **You.com** - 实时信息 + 新闻，追热点必备
7. **SearXNG** - 隐私优先，无追踪，完全免费！

## 💰 省钱小妙招

最绝的是有**智能缓存**！同样问题1小时内直接从缓存取，不花钱。

```bash
# 第一次：调用 API（花钱）
python3 scripts/search.py -q "AI 最新动态"

# 第二次：直接用缓存（免费）  
python3 scripts/search.py -q "AI 最新动态"
# 会显示："cached": true
```

## 🛠️ 怎么用？

简单到爆！一行命令搞定：
```bash
# 自动选最佳搜索引擎
python3 scripts/search.py -q "MacBook Pro M3 价格"

# 指定搜索引擎
python3 scripts/search.py -p serper -q "iPhone 16 参数"
python3 scripts/search.py -p tavily -q "深度学习教程"
```

## 🎯 什么时候最爽？

- **买东西**：查参数→比价格→看评测
- **写论文**：概念解释→找论文→找代码  
- **做竞品**：找对手→看相似→深度对比
- **隐私搜索**：不想被追踪？用它！
- **多语言**：中文、日文、德文随便搜

```json

  "query": "iPhone 16 价格",

```

## ⚡ 快慢对比

| 搜索引擎 | 速度 | 最擅长 | 适合场景 |
|---------|------|--------|----------|
| Serper | 飞快 | 准确 | 买东西、查信息 |
| Tavily | 很快 | 深度 | 学术、技术分析 |
| Exa | 一般 | 语义 | 找相似、公司研究 |
| SearXNG | 很快 | 免费 | 不想被追踪时 |

## 🤔 有什么坑？

- 需要配 API key（至少一个）
- 有些搜索引擎要花钱
- 首次设置有点麻烦（但有向导自动配置）

## 🔧 新手建议

先配 Serper（免费额度大），用熟了再加别的。配置文件支持自定义关键词，比如把"价格"强制导向 Serper，"教程"导向 Tavily。
