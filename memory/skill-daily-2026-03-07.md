### 技能简介
Agent Skills for Obsidian - 让 AI 助手学会操作 Obsidian，支持 Markdown、Bases、JSON Canvas 和 CLI 操作。

### 核心功能
1. **obsidian-markdown** - 创建和编辑 Obsidian Flavored Markdown（wikilinks、embeds、callouts、properties）
2. **obsidian-bases** - 创建和编辑 Obsidian Bases（视图、筛选、公式、汇总）
3. **json-canvas** - 创建和编辑 JSON Canvas 文件（节点、边、分组、连接）
4. **obsidian-cli** - 通过 Obsidian CLI 与 vault 交互
5. **defuddle** - 从网页提取干净 markdown，去除杂乱内容

### 安装方法
```bash
npx clawhub@latest install obsidian-skills
```

或手动复制到 `~/.openclaw/skills/`

### 适用场景
- 写周报、整理读书笔记
- 搭建知识库
- 生成符合 Obsidian 规范的内容
- 自动化管理笔记库
