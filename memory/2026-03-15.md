## Doc-to-PPT LLM 调用改造 (23:53)

### 改动
1. **llm_adapter.py** - 从 aiohttp 直接调用 → OpenAI SDK
   - 配置统一从 `~/.openclaw/agents/main/agent/models.json` 读取
   - 自动重试、超时处理、流式输出
   - 依赖：`openai>=2.28.0`

2. **思考模式控制**
   - 添加 `enable_thinking` 参数，默认 `False`
   - 通过 `extra_body: {"enable_thinking": false}` 传给智谱 API
   - 即使有思考过程也只取最终输出

### 效果
- 响应更快（不浪费 token 在思考过程）
- 输出更干净（只有最终结果）
- 配置统一管理
