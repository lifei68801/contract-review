## AI 热点日报任务修复

### 问题描述
三个 cron 任务报错 `Message failed`，但消息实际已发送成功：
- AI热点日报 - consecutiveErrors: 4
- 小红书爆款内容生成 - consecutiveErrors: 1  
- X平台AI热点Post生成 - consecutiveErrors: 1

### 根因分析
子任务 agent 调用 message 工具时参数名错误：
```json
// 错误
{"action":"send","channel":"qqbot","topic":"qqbot:c2c:xxx",...}

// 正确
{"action":"send","channel":"qqbot","to":"qqbot:c2c:xxx",...}
```

### 状态
- cron delivery 成功发送（`lastDelivered: true`）
- 内部 message 工具调用失败
- 任务 payload 已包含"不要调用 message 工具"指令，但 agent 仍尝试调用

### 已修复 (03:10 UTC)
更新三个任务的 payload，添加"绝对禁止"区块：
- AI热点日报 ✅
- 小红书爆款内容生成 ✅
- X平台AI热点Post生成 ✅

强化指令：
```
## 绝对禁止
- 【禁止】不要调用 message 工具！系统会自动发送你的输出到 QQ
- 【禁止】不要使用 topic 参数发送消息
- 【禁止】不要尝试任何形式的消息发送
```
